  - saját írott jegyzet
  - hivatalos Java API dokumentáció

  Minden más használata meg nem engedett segédeszköznek minősül.
  Korábban megírt forráskódok (állományok) sem használhatók!

* Meg nem engedett segédeszköz használata esetén a dolgozatra
  0 pontot kapnak.

* Az egyes feladatokat a megfelelő alkönyvtárban dolgozzák ki!
  Pl. az 1. feladat megoldásának a feladat1/ könyvtárban kell
  szerepelnie!

  Minden feladat mappája tartalmazza azokat a file-okat,
  amiket implementálnia kell. (Kivéve a 4-es feladatbeli
  TestMain.java forrásfájl) Például a főprogramot a
  Main.java állományban írjuk meg. Ha más fájlokban is
  dolgozni kell, akkor erre a feladat leírása külön kitér.

* A feladatokat az eddig tanult módszerekkel meg lehet oldani!
  Olyan megoldásokat várok, amikben az eddig tanultak
  köszönnek vissza! "Túl profi" megoldásokat nem fogok
  elfogadni! Ha valakinél ilyet látok, akkor azt fogom feltételezni,
  hogy az illető valamelyik weboldalról bemásolta a kódrészletet.

* Ha valaki mégis "profi" módszert akar alkalmazni, akkor
  halkan kérdezze meg az oktatót, hogy az adott technikát
  lehet-e használni.

* Ha valamelyik feladatnál fájlból kell olvasni, akkor az
  input állomány mellékelve lesz a feladat mappájában.

* Minden egyes feladat megoldása kész, komplett, minden mástól
  független megoldás legyen. Vagyis a projekt a feladat mappájába
  belépve egyszerűen fordítható és futtatható legyen.

  Példa:

  $ cd feladat1
  $ javac Main.java    # lefordul
  $ java Main          # lefut
